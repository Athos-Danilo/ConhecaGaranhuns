# Conheça Garanhuns

Projeto final da disciplina de Programação em Dispositivos Móveis.

## Integrantes
- Nome do estudante 1
- Nome do estudante 2

## Descrição
O aplicativo apresenta uma lista de pontos turísticos, culturais e naturais de Garanhuns. O usuário pode pesquisar os locais e abrir uma tela com informações detalhadas.

## Funcionalidades
- Listagem de locais
- Pesquisa por nome, categoria ou bairro
- Navegação entre telas
- Tela de detalhes
- Indicador de carregamento
- Tratamento de lista vazia

## Tecnologias
- JavaScript
- React Native
- Expo
- React Navigation

## Como executar

1. Instale as dependências:
```bash
npm install
```

2. Inicie o projeto:
```bash
npx expo start
```
